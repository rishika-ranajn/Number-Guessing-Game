**Number Guessing Game** is a C program that generates a random number between 1 and 100 and challenges the user to guess it. The program uses the rand() function along with a time-based seed to ensure a different number is generated each time it runs. The user is prompted to enter guesses repeatedly, and after each attempt, the program provides feedback indicating whether the guess is too high or too low. The loop continues until the correct number is guessed, at which point the program displays a success message along with the number of attempts taken. This program demonstrates basic C programming concepts such as loops, conditional statements, user input handling, and random number generation.
